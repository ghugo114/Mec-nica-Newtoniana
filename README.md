Comienza el programa que simula el movimiento de un proyectil con
agregado de un obstaculo contra el que el objeto rebota. Además se asume 
que a altura cero se encuentra el piso y que los bordes del gráfico simulan 
paredes contra las que el objeto también rebota.

Versión para leer desde terminal:
python3 proyectil_.py 9.8 0.1 35.0 20.0 15.0 10. 5.

g=9.8 (m/s²)
ga(rozamiento con aire)=0.1
ángulo inicial= 35.0 (grados)
velocidad inicial= 20.0 (m/s)
Tiempo animación= 15.0 (s)
Obstáculo distancia al orígen = 10 m
          altura = 5 m
          (internamente se dispuso un ancho de 0.05 m)
        

Versión que solicita parámetros al usuario
python3 proyectil.py
